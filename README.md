# Max Page Size

> Stop clicking "Show 60 per page." Max Page Size does it for you.

A browser extension for Chrome and Safari that automatically selects the highest available page size option on shopping sites, search results, and product lists. No more manual clicking — see more, scroll less, save time.

## ✨ Features

- 🚀 **Automatic** — Detects page size selectors on supported sites and selects the max
- 🌐 **Universal** — Works on hundreds of e-commerce, search, and listing sites
- ⚡ **Lightweight** — Tiny extension, no tracking, no background processes
- 🔒 **Privacy-first** — Runs entirely in your browser. No data collection, ever.
- 🎯 **Smart detection** — Recognizes common pagination patterns (Amazon, eBay, Shopify stores, etc.)
- ⚙️ **Customizable** — Configure which sites to enable/disable

## 📦 Installation

### Chrome
1. Visit the [Chrome Web Store](https://chrome.google.com/webstore/detail/icgnjkgcbcpbbfngehgpekggnebbnhec)
2. Click "Add to Chrome"
3. Done! Start browsing — Max Page Size kicks in automatically.

### Safari (macOS)
1. Install from the [Mac App Store](#) (link coming soon)
2. Open Safari → Preferences → Extensions
3. Enable "Max Page Size"

And hundreds more sites with standard pagination patterns.

## 🛠️ How It Works

1. You visit a page with a "Items per page" or "Show X per page" selector
2. Max Page Size detects the selector and reads the available options
3. It automatically selects the highest available number
4. The page reloads with more items — you scroll less, see more

No login, no account, no setup required.
